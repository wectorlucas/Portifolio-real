Sobre Mim

Sou um desenvolvedor entusiasta focado em criar soluções rápidas e personalizadas usando a plataforma Google AppScript. Minha grande habilidade é alavancar ferramentas de IA para construir aplicações complexas e eficientes, transformando ideias em realidade.

Especializo-me em criar web apps com AppScript, automações para Google Sheets e aplicativos com AppSheet. Além disso, construo páginas web estáticas usando GitHub Pages, incluindo portfólios, perfis pessoais, páginas de informação para empresas e sites semelhantes ao Linktree. O diferencial do meu trabalho é a personalização extrema para cada necessidade, tanto em funcionalidade quanto em design.
